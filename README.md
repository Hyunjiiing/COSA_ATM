## 로컬서버실행
가상환경 생성(파이썬버전주의)
```
$ conda create -n [이름] python=3.8
```

가상환경 activate 후 다운로드(아래 명령어 실행)!
```
$ conda env create -f conda_requirements.yaml
```

로컬서버 구동
```
$ uvicorn main:app --reload
```
